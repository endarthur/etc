Cover Generation

Covers are generated by image models via NanoGPT. The pipeline generates an illustration, not a finished cover — text and typography are your job, not the model's.
Different models produce different visual styles. Experiment.
Appends a style instruction to the cover prompt. See the Prompts tab to edit the full prompt.
Why no text on covers?
Image models can't reliably render text — they produce garbled letters, misspellings, and visual noise. Professional covers composite typography separately. The illustration is the foundation; the title is added later. If you want text, use an image editor or CSS overlay in the reader.
